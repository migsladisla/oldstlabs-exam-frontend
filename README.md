# Old St. Labs Technical Exam - Frontend

This is the client/frontend of the booking system application.

### Setup

1. Run `npm install`
2. Run `npm start`
3. Hit `localhost:8000/` to get started

### Tools/Packages Used

1. NodeJS/ExpressJS
2. TailwindCSS (handlebars template)
3. AlphineJS (datepicker)
4. jQuery
5. sweetalert2
6. validatorjs
7. axios
